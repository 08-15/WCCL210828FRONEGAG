Bienvenue dans votre nouvelle expérience de boîte aux lettres

Laissez votre assistant personnel vous expliquer les fonctions


Bonjour mon nom est Marie; En tant que nouvel assistant, permettez-moi de vous expliquer brièvement comment nous pouvons travailler ensemble de manière fluide et efficace.
Je suis toujours là pour vous et vous offre, entre autres, un APERÇU sécurisé des messages provenant d'expéditeurs inconnus. Vous pouvez reconnaître ces messages par le format de la ligne d'objet, qui commence toujours par mon nom, par exemple :

Marie : [info@e-mail.xing.com]

La ligne d'objet commence toujours par mon nom et contient l'adresse e-mail de l'expéditeur entre parenthèses telles que : [abc@unknown.com].

Par conséquent, mes messages sont faciles à repérer pour vous rassurer que ces aperçus sont sûrs et que le contenu est présenté sans fonctionnalités actives telles que des liens cliquables.
Vous savez donc déjà qu'il s'agit d'un message de quelqu'un qui n'est pas dans votre liste de contacts active. Par conséquent, vous devez les étudier attentivement (en particulier l'adresse e-mail de l'expéditeur) afin de décider s'il s'agit d'une lettre importante, ou simplement d'un spam, ou même d'une arnaque. Si vous n'êtes pas sûr et que le message ne semble pas important, vous feriez mieux de le supprimer. Le même expéditeur a la possibilité de vous écrire un autre e-mail avant qu'il ne soit automatiquement bloqué (avec le réglage standard).

Conseil : Un indicateur important est le format et le contenu de l'adresse e-mail de l'expéditeur : est-ce que cela semble normal et est-il lié au contenu du message ?

Si votre vérification aboutit à un résultat positif, vous aurez besoin du message d'origine (que j'ai mis en quarantaine pour votre protection) et souhaitez activer l'expéditeur afin que son prochain e-mail soit immédiatement envoyé dans votre boîte de réception.

Comment activer un expéditeur inconnu

Attention : lors de l'enregistrement de votre compte, j'ai créé deux sous-dossiers pour votre boîte mail :

L'un s'appelle "#Activate", l'autre "#Block-URL". Ces deux dossiers sont vos canaux de commande actifs.

Pour activer un nouvel expéditeur, il suffit de déplacer/déplacer le message d'aperçu associé dans le dossier #Aktivate (par exemple en glissant-déposant) et d'attendre que le message d'origine de l'expéditeur arrive.

Comment bloquer une URL

Certains spammeurs utilisent l'URL de leur adresse avec d'innombrables noms d'expéditeurs pour envoyer à plusieurs reprises des messages sous des noms différents. Si vous avez cette expérience et que vous souhaitez vous débarrasser rapidement de cette nuisance, vous pouvez bloquer l'URL de l'expéditeur. Il suffit de déplacer un seul aperçu d'un tel expéditeur vers #Block-URL. Déjà fait!

Remarque : Ne déplacez jamais un aperçu, par ex. d'un expéditeur Gmail, Yahoo, etc., à #Block-URL, car cela bloquera tous les utilisateurs de ce service. Si vous avez fait quelque chose de mal avec un bloc d'URL, pas de problème. Supprimez simplement l'aperçu correspondant du dossier #Block-URL.

Remarque : pour Microsoft Outlook, vous devez affecter ces dossiers afin de les rendre visibles. Veuillez noter notre vidéo d'instruction spéciale sur notre chaîne YouTube !

Autres fonctions : En plus de ces deux canaux directs, vous pouvez m'envoyer un e-mail pour toutes mes autres fonctionnalités en écrivant un e-mail à votre propre adresse e-mail (votre propre compte). Entrez simplement votre commande dans la ligne d'objet dans le format requis, qui est décrit ci-dessous.

Ici, chaque commande est expliquée dans une étape distincte (d'autres fonctions suivront).

Important : Les exemples ci-dessous montrent mon nom "Marie" et vous devrez le remplacer par le nom de votre assistant personnel qui apparaîtra dans votre message de bienvenue. Merci beaucoup.

Commandes MAIL :

Mettez la commande dans la ligne de référence comme indiqué ci-dessous et envoyez à votre propre adresse

Registre des commandes par courrier :

Marie:3

Nombre d'aperçus envoyés avant de bloquer un expéditeur (par exemple, 3 enverront 3 fois un aperçu avant de bloquer)
Remarque : le réglage 0 désactive complètement le blocage automatique, cela signifie que vous obtiendrez un aperçu pour tout message d'expéditeur inconnu et qu'aucun expéditeur ne sera bloqué !

How to register and activate email addresses from my email program?

• Select one of your accounts that is registered on your Spam-X eMailAgent and should have addresses added on, e.g. me@myaccount.com
• Make a "new mail" addressed to that account, in this example: me@myaccount.com
• Instruct your mail assistant (in this example it is Andre to register one or more addresses with the following command in the subject line of that mail:
Andre:register(new@contact.de, eric@malo.com)

• Please note: If you'll register more than one address, they must be comma separated with a space, as shown above
